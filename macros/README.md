# Макросы для модуля Blue Man Crafting

## create-recipe.js

Макрос для создания рецептов реагентов. Позволяет ГМ создавать свитки с рецептами, которые можно изучить в модуле крафта.

### Как использовать:

1. **Создайте макрос в Foundry:**
   - Откройте панель макросов
   - Нажмите "+" для создания нового макроса
   - Вставьте содержимое файла `create-recipe.js`
   - Назовите макрос "Создать рецепт"

2. **Запустите макрос:**
   - Нажмите на макрос для открытия окна создания рецепта

3. **Создание рецепта:**
   - Перетащите компоненты в зону "Исходные компоненты"
   - Укажите количество каждого компонента
   - Перетащите результат в зону "Результат"
   - Выберите категорию ингредиента или создайте новую
   - Выберите редкость свитка
   - Нажмите "Создать рецепт"

4. **Использование рецепта:**
   - Созданный свиток появится в world items
   - Перетащите свиток в окно крафта для изучения рецепта

### Функции:

- **Drag&Drop** для компонентов и результата
- **Управление количеством** компонентов (+/-)
- **Выбор категории** из существующих или создание новой
- **Система редкости** свитков (Обычный, Необычный, Редкий, Очень редкий, Легендарный)
- **Автоматическое создание** свитка с рецептом в world items

### Категории ингредиентов:

- Суспензия
- Эссенция
- Соль
- Зола
- Витриол
- Сублимат
- + Создание новых категорий

### Примечания:

- Используются стандартные иконки свитков (заглушка)
- Рецепты сохраняются во флагах предмета для последующего использования
- Макрос требует прав ГМ для создания предметов в world items
