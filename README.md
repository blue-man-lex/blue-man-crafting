# BG3 Crafting System

![Version](https://img.shields.io/badge/version-1.2.3-blue.svg)
![Foundry](https://img.shields.io/badge/Foundry-13%2B-green.svg)
![License](https://img.shields.io/badge/license-MIT-orange.svg)

**BG3 Crafting System** - это комплексный модуль крафта для Foundry VTT, вдохновленный системой создания предметов из Baldur's Gate 3. Модуль поддерживает 9 категорий ремесел, расходуемые инструменты с системой зарядов, интеллектуальные подсказки и автоматически определяет нужные наборы для каждого типа крафта.

## ✨ Основные возможности

### 🧪 **9 категорий ремесел**
- **Ингредиенты** - суспензии, эссенции, соли, золы, купоросы, сублиматы
- **Алхимия** - зелья, эликсиры, гранаты, масла и яды
- **Кузнечное дело** - оружие, доспехи, инструменты
- **Ювелирное дело** - огранка камней, чародейская пыль
- **Кожевничество** - кожаные доспехи, дубление кожи
- **Кулинария** - рационы, пиры
- **Ткачество** - тканевые доспехи, вышивка
- **Начертание** - свитки заклинаний, чернила
- **Пользовательские** - настраиваемые категории для ГМ

### 🔧 **Система инструментов**
- **Расходуемые наборы** - 10 использований на каждый инструмент
- **Автоудаление** - инструменты автоматически удаляются при истечении зарядов
- **Проверка наличия** - система проверяет нужный набор для конкретного типа крафта
- **Визуальные индикаторы** - зеленая/красная подсветка доступности инструментов

### 📜 **Умные рецепты**
- **Динамическое создание** - рецепты можно добавлять через предметы-свитки или макросы
- **Автоматические категории** - система создает категории на основе папок
- **6 типов ингредиентов** - суспензии, эссенции, соли, золы, купоросы, сублиматы
- **Гибкая настройка** - поддержка пользовательских рецептов и категорий
- **Fallback категория** - "Пользовательские" → "Без категории" для кастомных рецептов

### 🎮 **Премиальный интерфейс**
- **Аккордеон** - разворачивающиеся категории и подкатегории
- **Drag & Drop** - перетаскивание предметов и изучение рецептов
- **Мультиплеер** - полноценная поддержка сетевой игры через socketlib
- **Подсветка редкости** - цветовая индикация Common, Uncommon, Rare, Very Rare, Legendary, Artifact
- **Интеллектуальные подсказки** - детальная информация о предметах при наведении
- **Анимации** - плавные переходы и эффекты в стиле BG3

### 🎨 **Визуальные особенности**
- **Градиенты BG3** - премиальный дизайн с многократными градиентами
- **Цветовая схема** - золотые и медные оттенки в стиле Baldur's Gate 3
- **Подсветка редкости предметов**:
  - **Common** - серый/белый
  - **Uncommon** - зеленый с свечением
  - **Rare** - голубой с свечением
  - **Very Rare** - фиолетовый/розовый с свечением
  - **Legendary** - золотой с интенсивным свечением
  - **Artifact** - оранжевый/красный с ярким свечением

## 🚀 Быстрый старт

### Установка
1. Скачайте модуль с [GitHub Releases](https://github.com/blue-man-lex/blue-man-crafting/releases)
2. Распакуйте в `modules/blue-man-crafting`
3. Активируйте в Foundry VTT
4. Убедитесь что модуль **socketlib** установлен и активирован

### Первое использование
1. Создайте персонажа или выберите существующего
2. Нажмите на иконку **🧪 Алхимия** в шапке листа персонажа
3. Откроется окно крафта с доступными рецептами

### Изучение рецептов
1. **Через свитки** - перетащите свиток рецепта в зону "Изучить рецепт"
2. **Через макросы** - используйте макросы создания рецептов
3. **Автоматически** - рецепты автоматически добавляются в соответствующие категории

## 🛠️ Создание пользовательских рецептов

### Способ 1: Через макросы (рекомендуется)

#### Макрос создания простого рецепта
```javascript
// Выполните этот макрос в Foundry
const recipeData = {
    name: "Зелье лечения",
    type: "potions", // подкатегория
    ingredients: [
        { type: "category", categoryId: "essence", qty: 1 },
        { type: "category", categoryId: "suspension", qty: 1 }
    ],
    result: { 
        uuid: "Compendium.dnd5e.items.Item.sB7E9B0MELb8jZs", 
        qty: 1 
    },
    categoryId: "alchemy", // основная категория
    subcategoryId: "potions" // подкатегория
};

// Создаем предмет-свиток с рецептом
const scrollData = {
    name: "Чертеж: Зелье лечения",
    type: "consumable",
    img: "icons/sundries/scrolls/scroll-bound-white-blue.webp",
    system: {
        description: {
            value: "<p>Рецепт создания зелья лечения</p>"
        },
        consumableType: "scroll"
    },
    flags: {
        "blue-man-crafting": {
            recipe: recipeData
        }
    }
};

// Создаем свиток в инвентаре выбранного персонажа
const actor = game.user.character;
if (actor) {
    await actor.createEmbeddedDocuments("Item", [scrollData]);
    ui.notifications.info("Свиток рецепта создан!");
}
```

#### Макрос создания сложного рецепта

### Способ 2: Через готовые макросы модуля

Модуль включает два готовых макроса:

1. **create-recipe.js** - создание рецептов с выбором категории
2. **create-final-recipe.js** - создание финальных рецептов

#### Использование готовых макросов:
1. Откройте вкладку **Macro** в Foundry
2. Создайте новый макрос
3. Вставьте код из файла макроса
4. Запустите макрос
5. Заполните форму рецепта
6. Выберите категорию и подкатегорию
7. Рецепт будет создан как предмет-свиток

### Способ 3: Через редактор рецептов (для ГМ)

1. Откройте **Настройки игры** → **Модули**
2. Найдите **BG3 Crafting System** → **Редактор рецептов**
3. Нажмите **"Добавить рецепт"** и заполните форму
4. Сохраните изменения

## 🌐 Мультиплеер функции

### 📡 Система socketlib

Модуль использует **socketlib** для мультиплеер взаимодействия:

#### Изучение рецептов в реальном времени
```javascript
// Автоматическая синхронизация при изучении рецепта
socket.executeForEveryone("updateCraftingUI", {
    recipeId: recipeId,
    isKnown: true
});

// Обновление интерфейса конкретного игрока
socket.executeAsUser("updateCraftingUI", userId, {
    recipeId: recipeId,
    isKnown: false
});
```

#### Обработчики сокетов
- **learnRecipeFromScroll** - изучение рецепта из свитка
- **recipeLearnedFromScroll** - подтверждение изучения
- **updateCraftingUI** - обновление интерфейса

### 🔄 Синхронизация данных

- **Мгновенное обновление** - интерфейсы всех игроков обновляются при изменениях
- **ГМ-контроль** - ГМ управляет доступом к рецептам в реальном времени
- **Сохранение состояния** - рецепты сохраняются в флагах персонажей
- **Резервное копирование** - автоматическое сохранение настроек нужного набора

## 🐛 Устранение неполадок

### 🔍 Частые проблемы

**❌ Рецепты не отображаются:**
- Проверьте наличие нужного инструмента для ремесла
- Убедитесь что рецепт изучен (для игроков)
- Проверьте правильность UUID в рецепте
- Проверьте что модуль socketlib активирован

**❌ Инструменты не работают:**
- Убедитесь что инструмент в инвентаре персонажа
- Проверьте количество зарядов (должно быть > 0)
- Проверьте соответствие инструмента типу крафта
- Проверьте UUID инструмента

**❌ Мультиплеер не работает:**
- Убедитесь что модуль **socketlib** установлен и активирован
- Проверьте что все игроки используют одну версию модуля
- Перезапустите Foundry VTT

**❌ Подсказки не отображаются:**
- Проверьте консоль на ошибки JavaScript
- Убедитесь что CSS стили загружены корректно
- Проверьте что у предметов есть корректные UUID

**❌ Ошибки в консоли:**
- Проверьте наличие модуля **socketlib**
- Убедитесь что версия Foundry VTT 13+
- Проверьте что все компендиумы модуля загружены
- Перезагрузите модуль через настройки

- **spell-scrolls (свитки заклинаний)** - магические свитки
