# BG3 Crafting System

![Version](https://img.shields.io/badge/version-1.1.0-blue.svg)
![Foundry](https://img.shields.io/badge/Foundry-13%2B-green.svg)
![License](https://img.shields.io/badge/license-MIT-orange.svg)

**BG3 Crafting System** - это комплексный модуль крафта для Foundry VTT, вдохновленный системой создания предметов из Baldur's Gate 3. Модуль поддерживает 6 категорий ремесел, расходуемые инструменты с системой зарядов и автоматически определяет нужные наборы для каждого типа крафта.

## ✨ Основные возможности

### 🧪 **6 категорий ремесел**
- **Алхимия** - зелья, эликсиры, гранаты, масла и яды
- **Кузнечное дело** - оружие, доспехи, инструменты
- **Ювелирное дело** - огранка камней, чародейская пыль
- **Кожевничество** - кожаные доспехи, дубление
- **Кулинария** - рационы, пиры
- **Ткачество** - тканевые доспехи, вышивка

### 🔧 **Система инструментов**
- **Расходуемые наборы** - 10 использований на каждый инструмент
- **Автоудаление** - инструменты автоматически удаляются при истечении зарядов
- **Проверка наличия** - система проверяет нужный набор для конкретного типа крафта
- **Визуальные индикаторы** - зеленая/красная подсветка доступности инструментов

### 📜 **Умные рецепты**
- **Динамическое создание** - рецепты можно добавлять через предметы-свитки
- **Автоматические категории** - система создает категории на основе папок
- **Ингредиенты** - 6 типов базовых компонентов (суспензии, соли, золы, купоросы, сублиматы, эссенции)
- **Гибкая настройка** - поддержка пользовательских рецептов и категорий

### 🎮 **Удобный интерфейс**
- **Аккордеон** - разворачивающиеся категории и подкатегории
- **Drag & Drop** - перетаскивание предметов и изучение рецептов
- **Мультиплеер** - полноценная поддержка сетевой игры через socketlib
- **Премиальный дизайн** - интерфейс в стиле BG3 с градиентами и анимациями

## 🚀 Быстрый старт

### Установка
1. Скачайте модуль с [GitHub Releases](https://github.com/blue-man-lex/blue-man-crafting/releases)
2. Распакуйте в `modules/blue-man-crafting`
3. Активируйте в Foundry VTT

### Первое использование
1. Создайте персонажа или выберите существующего
2. Нажмите на иконку **🧪 Алхимия** в шапке листа персонажа
3. Откроется окно крафта с доступными рецептами

## 🛠️ Создание пользовательских рецептов

### Способ 1: Через предмет-свиток (рекомендуется)

1. **Создайте структуру папок** во вкладке Items:
   ```
   📁 Крафт/
     📁 Зачарование/
       📁 Жезлы/
         📄 Огненный жезл
   ```

2. **Получите UUID предметов**:
   - Правый клик на предмет → Copy UUID
   - Сохраните UUID результата и ингредиентов

3. **Создайте свиток рецепта**:
   ```javascript
   // Выполните этот макрос в Foundry
   const recipeData = {
       name: "Огненный жезл",
       type: "wands",
       ingredients: [
           { type: "item", uuid: "UUID_ДЕРЕВЯННОЙ_ПАЛКИ", qty: 3 },
           { type: "item", uuid: "UUID_РУБИНА", qty: 1 },
           { type: "category", categoryId: "essence", qty: 1 }
       ],
       result: { uuid: "UUID_ОГНЕННОГО_ЖЕЗЛА", qty: 1 }
   };
   
   const scroll = game.items.getName("Чертеж: Огненный жезл");
   await scroll.setFlag("blue-man-crafting", "recipe", recipeData);
   ```

4. **Используйте свиток**:
   - Выдайте свиток персонажу
   - Перетащите его в зону "Изучить рецепт"
   - Рецепт добавится в меню крафта

### Способ 2: Через редактор рецептов (для ГМ)

1. Откройте **Настройки игры** → **Модули**
2. Найдите **BG3 Crafting System** → **Редактор рецептов**
3. Нажмите **"Добавить рецепт"** и заполните форму
4. Сохраните изменения

## 🎯 Администрирование

### 📊 Управление рецептами

**Редактор рецептов** доступен в настройках модуля:
- **Просмотр всех рецептов** - базовых и пользовательских
- **Добавление/удаление** - управление рецептами
- **Категории** - создание новых категорий и подкатегорий
- **Импорт/экспорт** - резервное копирование и передача рецептов

### 🛡️ Управление доступом

**Для ГМ доступно:**
- **Переключение видимости** рецептов для игроков
- **Массовые операции** - показать/скрыть все рецепты
- **Выбор игроков** - управление рецептами конкретных игроков
- **Сброс настроек** - очистка пользовательских данных

### 🔧 Консольные команды

### 💡 Как работает система зарядов

1. **Начальные заряды**: 10 использований
2. **Расход**: 1 заряд за успешный крафт
3. **Обновление**: Система автоматически обновляет количество зарядов
4. **Удаление**: При 0 зарядах инструмент удаляется с уведомлением
5. **Проверка**: Перед крафтом система проверяет наличие нужного набора

## 🐛 Устранение неполадок

### 🔍 Частые проблемы

**❌ Рецепты не отображаются:**
- Проверьте наличие нужного инструмента для ремесла
- Убедитесь что рецепт изучен (для игроков)
- Проверьте правильность UUID в рецепте

**❌ Инструменты не работают:**
- Убедитесь что инструмент в инвентаре персонажа
- Проверьте количество зарядов (должно быть > 0)
- Проверьте соответствие инструмента типу крафта

**❌ Ошибки в консоли:**
- Проверьте наличие модуля **socketlib**
- Убедитесь что версия Foundry VTT 13+
- Перезагрузите модуль через настройки

### 📝 Логирование
